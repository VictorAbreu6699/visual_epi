Gráfico Polar
Os gráficos de área polar são semelhantes aos gráficos de pizza, mas cada segmento tem o mesmo ângulo – o raio do segmento difere dependendo do valor.
Este tipo de gráfico geralmente é útil quando queremos mostrar dados de comparação semelhantes a um gráfico de pizza, mas também mostrar uma escala de valores para contexto.
Histograma
Um histograma é uma espécie de gráfico de barras que demonstra uma distribuição de frequências. No histograma, a base de cada uma das barras representa uma classe e a altura representa a quantidade ou frequência absoluta com que o valor de cada classe ocorre.
