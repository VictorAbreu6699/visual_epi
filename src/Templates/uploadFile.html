Importar bases de dados

Baixar modelo padrão

A plataforma Visual EPI possibilita o envio de bases de dados de quaisquer enfermidades. As fontes de dados devem possuir as seguintes informações/colunas para serem salvas na plataforma:

  • Identificação da enfermidade por nome/apelido
  • Municipio referente aos casos
  • Data em que foram coletados os casos
  • Número de casos relatados

Ao enviar a fonte de dados, a plataforma espera receber um arquivo no formato XLSX com um padrão predefinido de colunas. Esse padrão é acessível a partir do arquivo modelo padrão, que recomendamos utilizá-lo como estrutura para preencher as informações.
